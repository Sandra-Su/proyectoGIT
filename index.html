Intruccion Descripcion
git --version Nos muestra la version que tenemos instalada
git init inicializa el repositorio
git add Añade los archivos que hemos modificado o creado. Se puede especificar el nombre del archivo o poner un punto despues del add para que agregue todo
git commit -m "mensaje" Añade un comentario que explica los cambios que acabamos de hacer con el comando git add
git commit -am "mensaje" Junta la función del comando git add junto con git commit
git config –global user.name “nombre” Nos permite definir el nombre de nuestro usuario, para poder realizar los commits
git config –global user.email “correo electrónico” Nos permite definir el correo de nuestro usuario
